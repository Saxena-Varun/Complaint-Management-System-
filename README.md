# Complaint Management System

A PHP-based Online Complaint Management System.

## Features
- User Registration and Login
- User Complaint Management
- Admin Dashboard
- Integrated Grievance Tracking

## Installation
1. Clone the repository.
2. Import the `sqlfile/cms.sql` into your MySQL/MariaDB database.
3. Configure `admin/include/config.php` and `users/includes/config.php` with your database credentials.
4. Run standard PHP server (XAMPP/WAMP/LAMP).

## Credentials
Check `passwords.txt` for default admin and user credentials.
